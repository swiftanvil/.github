# SwiftAnvil

> **We forge Swift. You ship it.**

SwiftAnvil is a collection of developer tooling packages for the Apple ecosystem —
built with type safety, testability, and AI-friendly scaffolding in mind.

## 🔨 The Forge

| Package | Purpose | Status |
|---------|---------|--------|
| [`anvil-a11y`](https://github.com/swiftanvil/swiftanvil-anvil-a11y) | Type-safe accessibility identifiers | 🚧 In Progress |
| [`anvil-bench`](https://github.com/swiftanvil/swiftanvil-anvil-bench) | Performance benchmarking toolkit | 🚧 In Progress |
| [`anvil-strings`](https://github.com/swiftanvil/swiftanvil-anvil-strings) | Type-safe localization | 🚧 In Progress |
| [`swiftanvil-cli`](https://github.com/swiftanvil/swiftanvil-cli) | CLI scaffolding tool | 🚧 In Progress |
